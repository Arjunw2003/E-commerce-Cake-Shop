# 🎂 E-Commerce Cake Shop

An online Cake Shop web application developed using **PHP, MySQL, HTML, CSS, JavaScript, and Bootstrap**. This project allows users to browse cakes, add products to the cart, place orders, and manage customer information.

---

## 📌 Features

- 🏠 Home Page
- 🍰 Browse Different Cake Categories
- 🔍 Search Cakes
- 🛒 Add to Cart
- 💳 Buy Now
- ❤️ Wishlist
- 👤 User Registration & Login
- 📦 Order Management
- 📞 Contact Page
- ℹ️ About Us Page
- 📝 Blog Page
- 📱 Responsive Design

## 🚀 Installation

1. Clone the repository

```bash
git clone https://github.com/your-username/E-commerce-Cake-Shop.git
```

2. Move the project to the XAMPP `htdocs` folder.

3. Import the SQL database from the **db** folder into phpMyAdmin.

4. Start **Apache** and **MySQL** from XAMPP.

5. Open your browser and visit

```
http://localhost/E-commerce-Cake-Shop/
```
